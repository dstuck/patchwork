# Tile Upgrades

## Existing
- Pristine Bonus (gold): 5 bonus points if fully covering holes
- Lenient Bonus (blue): no penalty for missing holes

## Ideas
- Collectors Bonus (green): 2 bonus points for every collectible
- Time Bonus (purple?): Adds something like .4 seconds to the clock when placed
